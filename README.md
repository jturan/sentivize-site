# Sentivize Health Website

Static website for Sentivize Health, hosted on GitHub Pages.

## Overview

Medicare data sharing, simplified.

This is a static HTML/CSS/JS site containing:
- Home page with beta testing notice
- Privacy Policy
- Security Policy
- Terms & Conditions

## Deployment

This site is automatically deployed via GitHub Pages to [sentivize.com](https://sentivize.com).

## Local Development

Simply open `index.html` in a browser, or use a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`

## Updates

To update content:
1. Edit the HTML files
2. Commit and push changes
3. GitHub Pages will automatically deploy within 1-2 minutes
